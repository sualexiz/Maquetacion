Técnica para redefinir problemas

Diego Parra (2003) afirma que redefinir un problema nos puede ayudar a solucionarlo de una manera mucho más fácil y eficaz de lo que pensábamos en un principio. La redefinición es el arte de jugar con los problemas para encontrar múltiples caras diferentes a ellos, y darnos cuenta de cuál es el verdadero problema que podemos enfrentar de una manera cómoda, realista y eficaz.

¿Cómo aplicarla?

Parra propone formular los problemas con la siguiente estructura:

[Cómo] + [verbo en infinitivo] + [sustantivo] + [complemento]

Ejemplo: ¿Cómo reducir el tráfico en la ciudad? ¿Cómo reducir el tráfico en horas de congestión? ¿Cómo reducir el tráfico de taxis? ¿Cómo hacer que la gente camine más?
