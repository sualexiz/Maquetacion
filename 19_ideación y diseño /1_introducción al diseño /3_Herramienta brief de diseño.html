Herramienta brief de diseño

Para hacer el briefing no se necesita un formato predefinido, cada empresa tiene su propia manera de elaborarlo.

De todas formas, el diseñador deberá tener varias reuniones con la empresa o institución cliente, para la retroalimentación y aclaración de algunos puntos o factores que el brief no alcanza a expresar.

Para realizar un buen brief, se deben tener en cuenta las siguientes consideraciones:

  • Debe ser un documento concreto y sintetizado, sin mucho texto o explicaciones.
  • Debe ser realista.
  • No debe ser genérico.
  • No debe tener vocabulario confuso.
  • No debería servir para aplicar a cualquier producto, servicio o proceso.

El brief es un documento muy importante. Sin embargo, se debe hacer investigación de campo y tratar de entender en lo mayor posible al usuario final, para así poder lograr un diseño exitoso. Haz clic en el botón para descargar el formato de brief.
