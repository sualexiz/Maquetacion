Cuadro de validación

Una de las etapas más complejas para quién elabora un proyecto es cuando debe gestionar el modelo de negocio en la realidad y descubre que no todo lo pensado es cierto.

Es importante para el líder del proyecto planear, ejecutar y controlar las actividades. Para ello existe una herramienta denominada «cuadro de validación» que fue desarrollada por la empresa Lean Startup Machine.

  Ver enlace

El objetivo de la herramienta es ayudar a gestionar adecuadamente los experimentos, la manera cómo se van a realizar y controlarán, para saber cuáles hipótesis son válidas y cuáles no.

A continuación se presenta el cuadro de validación y la manera cómo debe usarse, es una herramienta de documentación y seguimiento de las acciones realizadas para encontrar clientes, una solución y un modelo de negocio que pueda ser válido.
